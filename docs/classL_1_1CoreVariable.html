Detailed Description

Implements a variable.

Like every core object, core variables are the essence of what they represent in the sense that contrary to Variable's, it posesses its attributes. CoreVariable's should never be exposed to the user. Only the Environment class shall be allowed for their creation. Environment is in charge of core variable's memory.
