Detailed Description

template<class Component>
class L::Vector< Component >

Implements a vector of components.

Vectors are a sort of factory for its components. It does not manage their storage. For instance, instead of creating several variables x_1, x_2, x_3, one shall create a VariableVector and ask for x(1), x(2) and x(3). The vector does nothing but calling the Variable's constructor with incrementing variables names.
