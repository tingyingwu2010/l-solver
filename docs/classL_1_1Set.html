Detailed Description

template<class Index, class CoreComponent, class Component, class ConstComponent>
class L::Set< Index, CoreComponent, Component, ConstComponent >

Represents a composition of components. Sets are responsible for the life and death of their components.

Template Parameters
Indexthe class used to represent an index for the components (e.g., storage by name -> std::string)
CoreComponentclass which implements the core components (see http://hlefebvr.github.io/posts/en_lsolver_modeling_milp.html)
Componentclass which implements the regular components (see http://hlefebvr.github.io/posts/en_lsolver_modeling_milp.html)
ConstComponentclass which implements the const components (see http://hlefebvr.github.io/posts/en_lsolver_modeling_milp.html)
