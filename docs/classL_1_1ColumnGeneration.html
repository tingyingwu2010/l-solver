Detailed Description

template<class Adapter_RMP>
class L::ColumnGeneration< Adapter_RMP >

Implements the generic column generation procedure.

Column generation may be applied in cases where the number of variables is very large compare to the number of constraints provided that one posesses a polyhedral description of the aforementioned columns. Formally, we may consider the following model

\begin{align*} \textrm{minimize } & c^Tx + d^Ty \\ \textrm{s.t. } & Ax \ge b \\ & \sum_{i=j}^N t_{ij}x_j + \sum_{j|y_j\in\mathcal Y} h_{ij}y_j \ge f_i \quad \forall i = 1,...,M \end{align*}

where $ Y$ is a polyhedron.
